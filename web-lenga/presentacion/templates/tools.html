Herramientas de Trabajo

Power BI

Permite crear visualizaciones interactivas y generar reportes automáticos, facilitando la toma de decisiones estratégicas basadas en datos procesados.

Python

Utilizado para desarrollar scripts de ETL, combinando tablas, conversión de fechas y la creación de gráficos y reportes de calidad de trozas.

Trello

Gestor de tareas que permite visualizar el estado de las tareas en curso y asegurarse de que el equipo cumple con los plazos establecidos.

GitHub

Utilizado para el control de versiones y colaboración en el código fuente del proyecto, asegurando trazabilidad y control de cambios.

Google Drive

Almacenamiento y compartición de documentos clave del proyecto, facilitando la colaboración en tiempo real.

Visual Studio Code

Entorno de desarrollo integrado utilizado para trabajar con scripts de Python y colaborar mediante GitHub.
